# WCQF Terminal · Mundial 2026 ### App móvil · Deploy en Cloudflare Pages via GitHub --- ## 🚀 Despliegue (paso a paso) ### PARTE 1 — Subir a GitHub 1. Ve a [github.com](https://github.com) → **New repository** 2. Nombre: `wcqf-app` (puede ser privado) 3. En tu computadora, entra a la carpeta descargada: ``` cd wcqf-netlify git init git add . git commit -m "WCQF app inicial" git remote add origin https://github.com/TU_USUARIO/wcqf-app.git git push -u origin main ``` --- ### PARTE 2 — Conectar con Cloudflare Pages 1. Ve a [pages.cloudflare.com](https://pages.cloudflare.com) → inicia sesión (gratis) 2. Click **Create a project → Connect to Git** 3. Selecciona tu cuenta GitHub y elige el repo `wcqf-app` 4. En **Build settings** deja todo vacío: - Framework preset: **None** - Build command: *(dejar en blanco)* - Build output directory: *(dejar en blanco)* 5. Click **Save and Deploy** → espera ~30 segundos 6. ¡Tienes URL pública! Ej: `wcqf-app.pages.dev` --- ### PARTE 3 — Variables de entorno Betfair (opcional) Sin esto la app funciona con cuotas estáticas de muestra. Con esto las cuotas se actualizan en vivo cada 5 minutos. 1. Cloudflare Pages → tu proyecto → **Settings → Environment Variables** 2. Agrega estas 3 variables (en **Production**): | Variable | Valor | |-----------------------|------------------------| | `BETFAIR_APP_KEY` | tu App Key de Betfair | | `BETFAIR_USERNAME` | tu usuario Betfair | | `BETFAIR_PASSWORD` | tu contraseña Betfair | 3. **Deployments → Create new deployment** (para que tome las variables) Para obtener tu App Key: [developer.betfair.com](https://developer.betfair.com) → crea app → tier Delayed (gratuito). --- ### PARTE 4 — Instalar en tu iPhone como app 1. Abre la URL en **Safari** (no Chrome) 2. Ícono de compartir → **"Agregar a inicio"** 3. Aparece en tu pantalla como app nativa --- ## 📁 Estructura del proyecto ``` wcqf-app/ ├── index.html ← App shell (PWA) ├── manifest.json ← Instalar como app ├── sw.js ← Offline / caché ├── _headers ← Cache-Control Cloudflare ├── css/app.css ├── js/ │ ├── data.js ← 48 equipos, resultados, calendario │ ├── engine.js ← Modelo Poisson-Elo + Monte Carlo │ ├── picks.js ← Picks con reglas de consistencia + Alpha │ ├── betfair.js ← Cliente Betfair (polling 5 min) │ └── app.js ← Controlador de pantallas └── functions/ └── api/ └── betfair-odds.js ← Proxy Betfair (Cloudflare Function) ``` --- ## Para actualizar datos (nuevos resultados, partidos, etc.) Solo edita `js/data.js`, haz commit a GitHub, y Cloudflare re-deploya automáticamente en ~20 segundos.
